# Snake Game

Snake game with SDL2 and Beginner friendly

# Usage

```bash
git clone https://github.com/fhawk12/snake-game.git
cd snake-game
make
./game
```

| Key   | Function   |
| ----- | ---------- |
| Up    | Go up      |
| Down  | Go down    |
| Left  | Go left    |
| Right | Go right   |
| Q     | Quit game  |
| R     | Reset game |
