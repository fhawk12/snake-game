# Snake Game

Snake game with SDL2 and Beginner friendly

# Usage

```bash
git clone https://github.com/fhawk12/snake-game.git
cd snake-game
make
./game
```

| Key   | Function   |
| ----- | ---------- |
| up    | go up      |
| down  | go down    |
| left  | go left    |
| right | go right   |
| q     | Quit game  |
| r     | Reset game |
